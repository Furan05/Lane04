# Copy & Voice — LANE 04

Extrait fidèle du case study **§02 Verbal Identity** (+ tags §06, VoiceOver §05).

## Ton — trois piliers

- **CHIRURGICAL** — Chaque mot est une donnée. On coupe les adjectifs, les encouragements, les points d'exclamation. Une phrase = une information.
- **IMMÉDIAT** — Verbes à l'impératif technique, résultats au présent. L'interface confirme en moins de trois mots : `PAYLOAD DELIVERED`.
- **BRUT** — Vocabulaire d'atelier et de terminal, jamais de marketing. Pas d'emoji, pas de gamification, pas de « bravo ». Le respect du coureur passe par la sobriété.

## Règle d'or

> L'**anglais technique** est réservé aux **commandes et statuts système** (mono, MAJUSCULES). Le **français** reste la langue de lecture : descriptions, aide, onboarding.

## Grille de conversion lexicale — product micro-copy

| DON'T — fitness générique | DO — LANE 04 |
|---|---|
| Créer une séance | `COMPILE PROTOCOL` |
| Sauvegarder | `COMMIT` |
| Envoyer à la montre | `INJECT PAYLOAD` |
| Séance envoyée ! 🎉 | `PAYLOAD DELIVERED` |
| Échauffement / Retour au calme | `WARM-UP` / `COOL-DOWN` |
| Historique des activités | `LOGS` |
| Réglages | `CONSOLE` |
| Mon profil | `OPERATOR` |
| Oups, une erreur est survenue… | `SYNC FAULT — RETRY` |

## Statuts système (`[BRACKET]`, mono, radius 0)

`[DRAFT]` · `[PAYLOAD READY]` · `[TX…]` / `[INJECTING xx%]` · `[SYNCED]` · `[SEARCHING]` · `[NO LINK]` · `[ACCESS DENIED]` · `[SYNC FAULT]` · `[IDLE]` · `[CALIBRATED]` · `[V1.0]` · `[30D]` · `[OUT OF RANGE]` · `[READY]`

Grammaire visuelle : pointillé = brouillon, contour = attente, aplat = confirmé, clignotement 1.2 s = faute.

## Tags de séance (filière, contour, couleur = température physiologique)

`[VMA]` → `[SEUIL]` → `[TEMPO]` → `[FARTLEK]` → `[RECUP]`
Du EMBER pur (VMA) vers le CRYO (récupération). Contour, jamais rempli sauf sélection.

## VoiceOver — français clair (le vocabulaire système est une couche *visuelle*)

VoiceOver parle un **français clair, sans jargon**. Aucun état n'est porté par la couleur seule — le mot est toujours présent.

| Élément à l'écran | Label VoiceOver |
|---|---|
| `INJECT PAYLOAD` | « Envoyer la séance vers l'Apple Watch » |
| `[PAYLOAD READY]` | « Séance prête à envoyer » |
| `[SYNC FAULT]` | « Échec de synchronisation. Réessayer. » |
| Stepper `6×` | « Répétitions : six. Ajustable. » |

## Manifestes (onboarding / éditorial — français)

- **001** — Nous ne vendons pas la motivation. Nous fournissons l'instrument.
- **002** — Ta séance n'est pas un contenu. C'est un protocole — et il t'appartient.
- **003** — Zéro abonnement, zéro flux, zéro bruit. Le chrono suffit.
- **004** — Le couloir 4 est celui du test. Entre, exécute, sors.

North star : **PRÉCISION SOUVERAINE.** Accroche produit : **COMPILE. INJECT. RUN.**
